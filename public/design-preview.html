✨ Design Features
  • Font Turtles: Playfair Display untuk heading dengan letter-spacing 1-3px
  • Responsive Design: Mobile-first dengan breakpoints di 480px, 768px
  • Card Hover Effect: Transform translateY(-8px) dengan smooth transition
  • Icon Animation: Float animation 3s infinite pada card icons
  • Gradient Backgrounds: Linear gradient untuk header dan overlay
  • Text Shadow: Multiple text shadows untuk depth dan readability
  • Component Images: Menggunakan bg_chatbot.png, bg_dinasti.png sebagai backgrounds
  • Mobile Optimization: Touch-friendly buttons dan responsive font sizes
